# Task Tracker App

A simple task tracker built using JavaScript, HTML and CSS.

## Features
- Add tasks dynamically
- Simple and clean interface
- JavaScript DOM manipulation
- Real-time UI updates without page refresh

## Technologies Used
- JavaScript
- HTML
- CSS

## What I Learned
This project helped me practise DOM manipulation, event listeners, and structuring JavaScript code for interactive user interfaces.

## Future Improvements
- Task deletion
- Task editing
- Local storage persistence
